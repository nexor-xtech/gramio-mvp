# 🤝 Contributing to Gramio

Thanks for your interest in building Gramio.

## 🚀 How to contribute

1. Pick an issue from the Issues tab
2. Comment on the issue saying you want to work on it
3. Fork the repository
4. Create a new branch
5. Build your feature
6. Submit a Pull Request

---

## 🧱 Stack
- React / Next.js
- Node.js
- Supabase / Firebase
- Tailwind CSS

---

## 🧠 Important
We are building an MVP. Keep things simple, clean, and focused.

No overengineering.
